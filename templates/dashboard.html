PUP MEDICS

The PUP Medical Directory & Information Contact System is a capstone project created by BSIT undergrads in 2023 as part of their graduation pre-requisites. This website contains a basic CRUD functionality that is meant to be used by nurses in the clinic in a desktop computer for document retrieval. The app offers a vastly superior experience as it contains the four main features of MEDICS, which is appointment setting, medicinal prescription and history, emergency reporting, and data analytics.
